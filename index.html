Blog: Manual de Assassinatos para Boas Garotas

O projeto é uma página web criada com HTML e CSS, destinada a exibir um blog dedicado à série "Manual de Assassinato para Boas Garotas" . O objetivo é criar um ambiente virtual onde seja possível compartilhar e discutir detalhes da série, fornecendo resumos de capítulos, análises de personagens, e insights sobre os mistérios e reviravoltas da trama. A página também incluirá uma seção com links para aquisição dos livros da série.

A estrutura do site é baseada em HTML, que define a semântica e o layout do conteúdo, enquanto o CSS é usado para estilizar a página visualmente, incluindo núcleos, tipografia e a disposição dos elementos.

O HTML organiza o conteúdo em um cabeçalho com o título do blog, seguido por uma divisão principal (.container) que apresenta o propósito do site e uma coleção de postagens sobre a série em uma seção flexível (.posts). Cada post é destacado em seu próprio contêiner (.post), mostrando uma imagem relacionada ao tema, detalhes sobre o capítulo ou personagem, e um link para continuar a leitura ou comprar o livro. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela.

O CSS personaliza o visual do site, usando variáveis ​​para núcleos, estilizando o texto com uma fonte importada do Google Fonts e aplicando um esquema de núcleos que transmite a atmosfera sombria e misteriosa da série, com botões interativos. O uso de HTML e CSS é importante pois permite criar um site acessível e esteticamente envolvente sem a necessidade de scripts complexos, com foco na usabilidade e na experiência da pessoa usuária. A escolha da tipografia e do esquema de núcleos reforça a sensação de suspense, incentivando a exploração dos mistérios e teorias dentro do blog.

Jogo de Ping Pong Online

Este projeto é um jogo interativo de pingue-pongue, permitindo ao usuário desafiar a "máquina" em uma disputa por pontuação. O objetivo é proporcionar uma experiência divertida e competitiva, onde os jogadores possam aprimorar suas habilidades e desafiar a IA para ver quem consegue alcançar a maior pontuação

A estrutura do jogo é baseada em HTML, CSS e JavaScript, onde o HTML define o layout do jogo e o CSS estiliza a página, enquanto o JavaScript controla a lógica do jogo. O jogador pode mover a raquete com o teclado para rebater a bola, enquanto o IA responde automaticamente com movimentos predefinidos, criando uma dinâmica de competição.

Cada elemento do jogo, como a raquete do jogador e da IA, a bola e a área de jogo, é controlado por scripts que definem seu comportamento. Eventos como o início do jogo, entrega do jogador ou o contato da bola com as raquetes desencadeiam ações específicas, como a mudança de direção da bola, atualização da pontuação e detecção de colisões. A física simples do jogo, como a velocidade e direção da bola, é ajustada dinamicamente para aumentar a dificuldade à medida que o jogador progride.

A importância das ferramentas utilizadas, como JavaScript para a lógica e manipulação de elementos HTML, reside na sua capacidade de criar uma experiência de jogo responsiva e envolvente. O uso de scripts para controlar os eventos em tempo real, juntamente com gráficos simplificados e movimentos fluidos, proporciona uma experiência de jogo divertida e interativa. O jogador é incentivado a participar ativamente da ação, tentando superar a IA em uma competição eletrizante de pingue-pongue.

Flashcards: Manual de Assassinato Para Boas Garotas

Este projeto cria uma experiência interativa sobre a série "Manual de Assassinato Para Boas Garotas" , permitindo aos usuários explorar informações sobre o enredo e os personagens da série por meio de flashcards dinâmicos. O objetivo é oferecer uma maneira envolvente e educativa de conhecer detalhes dos personagens, mistérios e temas centrais da trama.

A estrutura HTML do conteúdo da página, organizando-o em frascos como a tela inicial, uma área de visualização dos flashcards, e botões de navegação para que os usuários possam alternar entre as diferentes informações dos cartões. O CSS é utilizado para estilizar a página, aplicando um esquema de núcleos que reflete a atmosfera de suspense da série, além de um layout responsivo que garante uma experiência de uso agradável em diferentes dispositivos. O JavaScript é responsável pela interatividade, permitindo que os flashcards mudem de conteúdo com base nas interações do usuário, apresentando detalhes como detalhes dos personagens, suas motivações e pontos-chave do enredo.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta os flashcards em uma seção interativa (.flashcards). Cada flashcard é exibido em seu próprio contêiner (.card), mostrando uma breve descrição e uma imagem relacionada ao conteúdo do cartão. O design responsivo é garantido pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes tamanhos de tela, garantindo que os usuários possam acessar os flashcards tanto em dispositivos móveis quanto em desktops.

Os scripts JavaScript controlam a lógica do funcionamento dos flashcards, permitindo que os usuários avancem e retrocedam entre os cartões, além de alterar o conteúdo apresentado de forma dinâmica. Essa abordagem modular facilita atualizações futuras no conteúdo, permitindo adicionar novos flashcards conforme a série avançada ou novos livros são lançados. A combinação de HTML, CSS e JavaScript é essencial para criar uma experiência imersiva que incentiva a exploração dos detalhes da série de maneira interativa e divertida.
